# Code of Ethics

1. **Light before profit** – if it harms users, we don’t ship it.  
2. **Open by default** – closed doors must be justified in writing.  
3. **Minimum viable governance** – decisions at the lowest possible layer.  
4. **Carbon-negative goal** – net-zero by 2026, net-negative by 2027.  
5. **Leave no neighbour offline** – run at least one public RPC on every continent.

Violation reports: ethics@lightman-trust.io (handled by an external ombudsperson).
