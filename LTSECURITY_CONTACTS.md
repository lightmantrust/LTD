# LTD Security Contacts

## Responsible Disclosure
If you identify a security vulnerability, please report immediately.

- **Primary Contact**: security@lightmantrust.com
- **Secondary Contact**: ops@lightmantrust.com
- **Phone (Emergency)**: +1-555-SECURE

## Reporting Guidelines
1. Provide detailed description of the issue.
2. Include steps to reproduce, screenshots, and logs.
3. Do not publicly disclose until a fix is confirmed.
4. Await acknowledgment within 24 hours.

## Response Commitment
- Initial response: within 24 hours
- Resolution target: critical issues within 72 hours
- Security advisories documented in `docs/policies/SECURITY_ADVISORY_LOG.md`
